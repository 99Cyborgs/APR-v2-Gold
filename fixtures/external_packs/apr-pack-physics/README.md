# APR Physics Pack

This fixture pack demonstrates the APR v2 advisory pack contract.

- It is loaded only by explicit path.
- It declares supported domains up front.
- It can return `not_applicable`.
- It can add scoped warnings, signals, fatal-gate requests, and escalation requests.
- It does not rewrite core recommendation semantics.

The pack is intentionally narrow. It looks for theory-physics credibility surfaces such as formal objects, observable anchors, discriminating consequences, and limiting-case language.
