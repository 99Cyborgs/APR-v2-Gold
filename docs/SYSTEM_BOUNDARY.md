# System Boundary

## Inside APR v2 core

- Input normalization and schema validation
- Claim extraction and evidence-anchor indexing
- Article, claim, domain, and outlet-profile classification
- Reviewability, transparency, integrity, scientific-record, and venue routing
- Ranked rehabilitation planning
- Advisory-only pack execution and reporting
- Canonical record emission and markdown rendering

## Outside APR v2 core

- Journal submission workflows
- Reviewer management
- Live editorial prioritization
- Domain-specific reporting checklists as universal rules
- Autonomous misconduct findings
- Any renderer that mutates canonical meaning
