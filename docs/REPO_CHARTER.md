# Repo Charter

APR v2 exists to produce deterministic, inspectable manuscript-audit records. The repository boundary is intentionally narrow:

- One canonical contract surface is active at runtime.
- Core semantics stop at reviewability, scientific-record readiness, venue-profile routing, ranked rehabilitation, and scoped integrity handling.
- Domain rules are externalized to advisory packs.
- Calibration claims must be backed by the gold-set harness, not by prose.

This repo does not host submission automation, collaboration tooling, CRM features, or domain-specific pack doctrine in core.
